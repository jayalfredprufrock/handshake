## Installation

```sh
npm install @jayalfredprufrock/handshake typebox
```

The Hono adapter requires Hono as a peer dependency:

```sh
npm install hono
```

## 1. Define a Contract

A contract describes every endpoint in your API. Create a shared file that both your server and client will import.

```ts
// contract.ts
import { createContract } from "@jayalfredprufrock/handshake/contract";
import { Type } from "typebox";

export const contract = createContract("/api", {
  getUser: {
    method: "GET",
    path: "/users/:id",
    params: Type.Object({ id: Type.String() }),
    response: Type.Object({
      id: Type.String(),
      name: Type.String(),
      email: Type.String(),
    }),
  },

  listUsers: {
    method: "GET",
    path: "/users",
    response: Type.Array(
      Type.Object({
        id: Type.String(),
        name: Type.String(),
        email: Type.String(),
      }),
    ),
  },

  createUser: {
    method: "POST",
    path: "/users",
    body: Type.Object({
      name: Type.String(),
      email: Type.String(),
    }),
    response: Type.Object({
      id: Type.String(),
      name: Type.String(),
      email: Type.String(),
    }),
  },

  deleteUser: {
    method: "DELETE",
    path: "/users/:id",
    params: Type.Object({ id: Type.String() }),
    response: Type.Object({ id: Type.String() }),
  },
});
```

## 2. Create a Server

Use `implementContract` to bind handlers to the contract, then assemble the app with `createHonoApp`. Every endpoint must have a handler ‚Äî `implementContract` throws if any is missing.

```ts
// server.ts
import { serve } from "@hono/node-server";
import { implementContract, createHonoApp } from "@jayalfredprufrock/handshake/hono";
import { contract } from "./contract";

const users = new Map<string, { id: string; name: string; email: string }>();
let nextId = 1;

const module = implementContract(contract, {
  getUser: ({ params }) => {
    const user = users.get(params.id);
    if (!user) throw contract.error("NOT_FOUND", "user not found"); // requires a NOT_FOUND error in the contract
    return user;
  },
  listUsers: () => [...users.values()],
  createUser: ({ body }) => {
    const id = String(nextId++);
    const user = { id, ...body };
    users.set(id, user);
    return user;
  },
  deleteUser: ({ params }) => {
    users.delete(params.id);
    return { id: params.id };
  },
});

const app = createHonoApp([module]);
serve({ fetch: app.fetch, port: 3000 });
```

Handler inputs are fully typed ‚Äî `params`, `query`, `headers`, and `body` are inferred from the contract. Returning a value that doesn't match the response schema is a compile-time error.

## 3. Create a Client

The client is generated directly from the contract. Each endpoint becomes a typed method.

```ts
// client.ts
import { createFetchClient } from "@jayalfredprufrock/handshake/client";
import { contract } from "./contract";

const api = createFetchClient(contract, { baseUrl: "http://localhost:3000" });

const users = await api.listUsers();
const created = await api.createUser({ name: "Alice", email: "alice@example.com" });
const user = await api.getUser({ id: created.id });
await api.deleteUser({ id: created.id });
```

`fetch` defaults to `globalThis.fetch`. Method signatures adapt to the endpoint ‚Äî params first, body next, and an `options` object (`query`, `headers`, `request`) last.
